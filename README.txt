INSTRUCCIONES PARA CORRER LA PAGINA

1. Descomprime este ZIP.
2. Abre la carpeta "pagina_personal_eduardo_actualizada".
3. Da doble clic en index.html para abrirlo en tu navegador.

Forma recomendada con Visual Studio Code:
1. Abre VS Code.
2. Archivo > Abrir carpeta > selecciona esta carpeta.
3. Instala la extension "Live Server" si no la tienes.
4. Da clic derecho sobre index.html > Open with Live Server.

Archivos importantes:
- index.html: estructura de la pagina.
- styles.css: colores, diseno y responsividad.
- script.js: menu movil y animaciones.
- assets/Eduardo_Arevalo_CV_English.pdf: CV descargable en ingles.
- assets/llave_publica_eduardo.pem: llave publica descargable.

Notas:
- Los enlaces de Instagram y Facebook estan colocados como /3duarevalo. Si tus enlaces reales son otros, cambialos en la seccion "Mis redes sociales" dentro de index.html.
- La llave publica se copio exactamente como fue subida. Si necesitas reemplazarla, sustituye el archivo assets/llave_publica_eduardo.pem conservando el mismo nombre.
